## Web API

## 1.What is Web API? 
Framework to build HTTP-based services accessible via browsers, apps, or devices.

## 2.Difference between Web API and WCF? 
Web API is HTTP/REST focused; WCF supports multiple protocols like TCP, HTTP, and more.

## 3.What is REST? 
Architectural style using HTTP verbs for CRUD operations on resources.

## 4.HTTP verbs in Web API? 
GET, POST, PUT, DELETE, PATCH.

## 5.What is routing in Web API? 
Maps URL requests to controller actions.

## 6.What is attribute routing? 
Define routes using attributes directly on controllers/actions.

## 7.Difference between IHttpActionResult and HttpResponseMessage? 
IHttpActionResult simplifies responses; HttpResponseMessage gives full control over response details.

## 8.What is content negotiation? 
Process of selecting the best response format (JSON, XML) based on client request.

## 9.How to version a Web API? 
Via URL path, query string, or custom headers.

## 10.What is a message handler? 
Custom logic executed before/after HTTP request processing.

## 11.What is dependency injection in Web API? 
Inject services into controllers for better testability and loose coupling.

## 12.How to secure Web API? 
Use authentication (JWT, OAuth) and authorization (roles, claims).

## 13.Difference between PUT and PATCH? 
PUT replaces entire resource; PATCH updates partially.

## 14.What is Model Binding? 
Converts HTTP request data to action method parameters.

## 15.What is Exception Handling in Web API? 
Use Exception Filters or global error handling via ExceptionHandler.

## 16.How to call a Web API from client? 
Using HttpClient, AJAX, Postman, or browser.

## 17.What is CORS? 
Cross-Origin Resource Sharing allows cross-domain requests.

## 18.Difference between Web API and MVC? 
Web API returns data (JSON/XML); MVC returns views (HTML).

## 19.What is OData in Web API? 
Protocol for querying and manipulating RESTful data via standardized URL syntax.

## 20.What is self-hosting in Web API? 
Running API in a console or Windows service without IIS.
